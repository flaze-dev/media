# Flaze Media
A flaze media library for the web. Support for MediaStreams and MediaRecording. 

## Next.js
```bash 
# Install Next.js
$ yarn create next-app --typescript .
```

## Development and building
```bash 
# Start Next.js for development
$ yarn dev

# Build library
$ yarn build
```

## Publishing
```bash
$ yarn npm:login
$ yarn publish
```

## Setup
```bash
$ yarn install
$ yarn setup
```
